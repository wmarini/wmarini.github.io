I'm a senior software engineer working mostly in C++ on Linux backends and
embedded systems — bare metal and real-time kernels included. These days I
work full-time on the open source **GNU binutils-gdb** project, porting it to
the **HPE NonStop Server**.

Day-to-day languages: C++, C, Rust, Python, Lua, and more Bash than I'd like
to admit.

This blog is where I write down the things I keep re-deriving: porting notes,
debugger internals, toolchain quirks, and the occasional systems programming
rant.
